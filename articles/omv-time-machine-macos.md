<h2>Introduction</h2><p>If you use macOS, Time Machine is a built-in backup solution that's simple and reliable. OpenMediaVault (OMV) can serve as a Time Machine backup target, making it easy to back up all your Mac devices over the network to a Raspberry Pi or home server.</p><h2>What You'll Need</h2><ul><li>An OMV server or Raspberry Pi running OpenMediaVault</li><li>macOS computer with Time Machine</li><li>A shared data drive or storage location on OMV</li></ul><h2>Step 1: Install OMV-Extras and Enable SMB</h2><p>Ensure OMV-Extras is installed and go to <strong>Services &gt; SMB/CIFS</strong>. Enable the SMB service and apply the changes.</p><h2>Step 2: Create a Shared Folder for Backups</h2><ul><li>Go to <strong>Storage &gt; File Systems</strong> and mount your data drive if not already mounted.</li><li>Then go to <strong>Access Rights Management &gt; Shared Folders</strong> and create a folder named <code>timemachine</code>.</li></ul><h2>Step 3: Configure SMB Share for Time Machine</h2><ul><li>Go to <strong>Services &gt; SMB/CIFS &gt; Shares</strong></li><li>Click <strong>Add</strong>, select the <code>timemachine</code> folder</li><li>Enable <strong>Time Machine support</strong></li><li>Set <strong>Public</strong> to <code>No</code> and give access to your macOS user</li></ul><h2>Step 4: Add macOS User Account</h2><ul><li>Under <strong>Access Rights Management &gt; User</strong>, create a new user with a password</li><li>Ensure this user is added to the <code>users</code> group</li></ul><h2>Step 5: Connect from macOS</h2><ol><li>On your Mac, open <code>Finder &gt; Go &gt; Connect to Server</code></li><li>Type: <code>smb://[OMV-IP]/timemachine</code> and connect with your OMV user credentials</li><li>Once mounted, open <code>System Preferences &gt; Time Machine</code></li><li>Click <strong>Select Disk</strong> and choose the <code>timemachine</code> share</li></ol><h2>Tips</h2><ul><li>Limit the share size by adjusting disk quotas under shared folder settings</li><li>Ensure your OMV server stays online during scheduled backups</li><li>Monitor space usage to avoid failed backups</li></ul><h2>Conclusion</h2><p>This setup lets you use OpenMediaVault as a native backup target for all your Mac devices, giving you full control over storage and peace of mind.</p>
