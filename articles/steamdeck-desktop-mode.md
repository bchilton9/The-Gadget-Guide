<h2>What is Desktop Mode?</h2>
  <p>The Steam Deck runs on SteamOS, a Linux-based operating system. While the default interface (Game Mode) is designed for gaming, you can switch to Desktop Mode for full PC functionality. This gives you access to a full KDE Plasma desktop, allowing you to install apps, manage files, browse the web, and use your Deck like a regular Linux PC.</p>

  <h2>How to Switch to Desktop Mode</h2>
  <ol>
    <li>Press the <strong>Steam</strong> button to open the menu.</li>
    <li>Scroll down and select <strong>Power</strong>.</li>
    <li>Choose <strong>Switch to Desktop</strong>.</li>
  </ol>
  <p>After a few seconds, your Deck will switch over to the KDE Plasma desktop environment.</p>

  <h2>Basic Controls in Desktop Mode</h2>
  <ul>
    <li><strong>Left Trackpad:</strong> Mouse pointer</li>
    <li><strong>Right Trackpad:</strong> Scroll</li>
    <li><strong>Right Trigger:</strong> Left click</li>
    <li><strong>Left Trigger:</strong> Right click</li>
    <li><strong>Steam + X:</strong> On-screen keyboard</li>
  </ul>

  <h2>Installing Software</h2>
  <p>Use the <strong>Discover</strong> app (software center) to browse and install apps. You can also use the terminal (Konsole) to install packages manually using pacman:</p>
  <pre><code>sudo pacman -S packagename</code></pre>

  <h2>Accessing Files</h2>
  <p>Use the Dolphin file manager to access your files, SD card, and USB drives. You can also drag and drop files from external drives or over a network.</p>

  <h2>Recommended Apps</h2>
  <ul>
    <li><strong>Firefox or Chromium:</strong> Web browsing</li>
    <li><strong>KDE Partition Manager:</strong> Disk tools</li>
    <li><strong>Flatseal:</strong> Manage Flatpak permissions</li>
    <li><strong>Heroic Games Launcher:</strong> Play Epic/GOG games</li>
    <li><strong>Lutris:</strong> Run Windows games and apps</li>
  </ul>

  <h2>Switching Back to Game Mode</h2>
  <p>Double-click the <strong>Return to Gaming Mode</strong> icon on the desktop, or restart the device.</p>

  <h2>Conclusion</h2>
  <p>Desktop Mode dramatically expands the power of your Steam Deck. Whether you want to install emulators, tweak system settings, or use it like a full PC, Desktop Mode is your gateway to advanced usage.</p>
