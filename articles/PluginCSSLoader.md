<h2>Plugin: CSS Loader</h2>
<p><strong>CSS Loader</strong> is a Decky Loader plugin that lets you fully customize the appearance and layout of your Steam Deck's Game Mode library using custom CSS themes.</p>

<p>Don’t have Decky Loader yet? <a href="index.php#DeckyLoader">Click here to install it first.</a></p>

<hr>

<h3>1. Why Use CSS Loader?</h3>
<img src="/images/IMG_1333.png" alt="Custom Steam Deck Theme Example">
<p>This is one of the most powerful and popular Decky plugins. It allows you to completely transform your Steam Deck’s interface to suit your personal style.</p>

<img src="/images/IMG_1334.png" alt="CSS Loader Plugin Card">
<p>To install CSS Loader, open the Decky Loader market, scroll or search for <strong>CSS Loader</strong>, then click <strong>Install</strong>.</p>

<hr>

<h3>2. Accessing CSS Loader</h3>
<img src="/images/IMG_1328.png" alt="Decky Loader Menu">
<p>Once installed, press the <strong>three dots button</strong> on the right. Select the <strong>Decky plug icon</strong> at the bottom, then choose <strong>CSS Loader</strong> from the list.</p>

<img src="/images/IMG_1336.png" alt="CSS Loader Interface">
<p>Inside CSS Loader:</p>
<ul>
  <li>The <strong>settings cog</strong> (top right) lets you manage installed themes.</li>
  <li>The icon to the left of the settings cog is the <strong>Download Themes</strong> button.</li>
  <li>The <strong>Selected Profile</strong> drop-down allows you to save and switch between custom layout profiles.</li>
  <li>Below is a list of your installed themes, each with a toggle to enable/disable and sometimes a dropdown for theme-specific settings.</li>
</ul>

<img src="/images/No-Image.png" alt="Theme Preview Page">
<p>Clicking the download icon opens a theme gallery with previews and descriptions. Select the theme you want and install it. Then toggle it on in the main CSS Loader interface and customize any settings as desired.</p>

<hr>

<h3>3. Notes & Tips</h3>
<p>There are hundreds of community-made themes—far too many to list here. Feel free to explore and experiment. You can always disable or uninstall a theme you don't like.</p>

<p>Some themes are bundled with other sub-themes or components to create the full visual experience intended by the creator.</p>
