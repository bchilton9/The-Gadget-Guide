<h2>Decky Loader</h2>
<p><strong>Decky Loader</strong> is a program for installing 3rd party plugins into Game Mode on your Steam Deck. It enables a wide range of quality-of-life enhancements and customization options.</p>

<p>Special thanks to the Decky Loader team for making this possible.</p>

<hr>

<h3>1. Switch to Desktop Mode</h3>
<p>Press the <strong>Steam</strong> button, select <strong>Power</strong>, then choose <strong>Switch to Desktop</strong>.</p>

<h3>2. Download the Installer</h3>
<p>Visit the Decky Loader GitHub page:</p>
<p><a href="https://github.com/SteamDeckHomebrew/decky-loader" target="_blank">https://github.com/SteamDeckHomebrew/decky-loader</a></p>
<p>Click the large download button:</p>
<img src="/images/IMG_1323.png" alt="Download Decky Loader">

<h3>3. Run the Installer</h3>
<p>Open the <strong>Discover</strong> file browser and select <strong>Downloads</strong> from the left-hand panel.</p>
<img src="/images/IMG_1324.png" alt="File Browser - Downloads Folder">
<p>Double-click <code>decky_installer.desktop</code>. When prompted, choose <strong>Execute</strong>.</p>
<img src="/images/IMG_1325.png" alt="Execute Installer Prompt">
<p>You’ll be prompted for your <strong>SUDO password</strong>. Press the <strong>X button</strong> to bring up the keyboard and enter your password.</p>
<p>Don’t have a SUDO password yet? <a href="index.php#SUDOPassword">Set a SUDO password here.</a></p>
<img src="/images/IMG_1326.png" alt="SUDO Password">

<h3>4. Install Decky Loader</h3>
<p>When the installer opens, choose <strong>Install the latest release</strong> and click <strong>OK</strong>. If you already have it installed, you’ll see an option to update instead.</p>
<img src="/images/IMG_1327.png" alt="Decky Installer Options">
<p>After the installation completes, click <strong>OK</strong> and return to Game Mode.</p>

<hr>

<h3>5. Using Decky in Game Mode</h3>
<img src="/images/IMG_1328.png" alt="Decky in Game Mode">
<p>In Game Mode, press the <strong>three dots button</strong> on the right. The bottom icon that looks like a plug is Decky.</p>
<p>In the top-right of the Decky menu, you’ll find:</p>
<ul>
  <li><strong>Market icon</strong>: Browse and install plugins</li>
  <li><strong>Settings cog</strong>: Update or uninstall plugins</li>
</ul>
<img src="/images/IMG_1329.jpeg" alt="Decky Plugin Market">

<h3>6. Installing Plugins</h3>
<p>Open the Market and click the <strong>Install</strong> button next to any plugin you want to use.</p>

<hr>

<h3>Troubleshooting</h3>
<p>If Decky Loader disappears after a system update, restart your system first. If it doesn’t return, re-run the installer and select <strong>Update to the latest release</strong>.</p>
