<h2>How to Set Up Lutris on Steam Deck</h2>

  <p>Lutris is a powerful game launcher for Linux that allows you to run games from Epic Games, GOG, Battle.net, Origin, Ubisoft Connect, and more — all in one interface. On Steam Deck, it unlocks a huge world of non-Steam gaming.</p>

  <h3>What is Lutris?</h3>
  <p>Lutris is an open-source game manager that integrates with Wine, Proton, and various emulators. It simplifies the process of installing games and managing launchers that aren’t natively supported by Steam.</p>

  <h3>Step 1: Switch to Desktop Mode</h3>
  <ol>
    <li>Press the <strong>Steam</strong> button.</li>
    <li>Go to <strong>Power → Switch to Desktop</strong>.</li>
  </ol>

  <h3>Step 2: Install Lutris from Discover</h3>
  <ol>
    <li>Open the <strong>Discover Store</strong> (shopping bag icon in the taskbar).</li>
    <li>Search for <strong>Lutris</strong>.</li>
    <li>Click <strong>Install</strong>.</li>
  </ol>

  <p>You now have Lutris installed on your system. You can launch it from the Applications Menu under "Games."</p>

  <h3>Step 3: Install Required Runtimes</h3>
  <p>To ensure maximum compatibility, you may be prompted to install Wine, DXVK, or other dependencies. Lutris can auto-install these for you.</p>

  <ul>
    <li>If prompted, allow Lutris to install <strong>Wine</strong> and related components.</li>
    <li>Go to <strong>Preferences → Runners</strong> to enable support for specific platforms (e.g. Epic, GOG, etc.).</li>
  </ul>

  <h3>Step 4: Sign In to Your Accounts</h3>
  <ul>
    <li>Epic Games: Click the Epic icon in Lutris and sign in via the web.</li>
    <li>GOG: Connect your GOG account under preferences.</li>
    <li>Other launchers (Battle.net, Ubisoft): Install them manually via Lutris scripts.</li>
  </ul>

  <p>Many installers are available directly from the Lutris website at <a href="https://lutris.net/games/" target="_blank">lutris.net/games</a>.</p>

  <h3>Step 5: Add Lutris to Steam</h3>
  <ol>
    <li>Open <strong>Steam (in Desktop Mode)</strong>.</li>
    <li>Click <strong>Add a Game → Add a Non-Steam Game</strong>.</li>
    <li>Select <strong>Lutris</strong> from the list.</li>
    <li>Click <strong>Add Selected Programs</strong>.</li>
  </ol>

  <p>This lets you launch Lutris directly from Game Mode!</p>

  <h3>Step 6: Return to Game Mode and Launch Lutris</h3>
  <ol>
    <li>Return to Game Mode by clicking <strong>Return to Gaming Mode</strong> on the desktop.</li>
    <li>Find Lutris in your Steam Library under <strong>Non-Steam Games</strong>.</li>
    <li>Launch it and start installing your games!</li>
  </ol>

  <h3>Troubleshooting Tips</h3>
  <ul>
    <li><strong>Controller doesn’t work?</strong> Use a keyboard and mouse in Desktop Mode to configure launch options.</li>
    <li><strong>Game won’t start?</strong> Try changing the Wine version under the game’s settings in Lutris.</li>
    <li><strong>Epic not syncing?</strong> Make sure you logged in and allowed access when prompted in your browser.</li>
  </ul>

  <h3>Conclusion</h3>
  <p>Lutris is a fantastic way to manage all your non-Steam games on the Steam Deck. Once configured, it works seamlessly in Game Mode and gives you access to a huge library of games beyond what Steam offers.</p>
