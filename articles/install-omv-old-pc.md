<h2>Introduction</h2><p>If you have an old desktop or laptop gathering dust, you can convert it into a fully functional Network Attached Storage (NAS) using OpenMediaVault (OMV). OMV is a Debian-based OS designed to turn any hardware into a centralized file server, media server, backup station, or Docker host.</p><h2>Why Use an Old PC?</h2><ul><li>Low cost – reuse existing hardware</li><li>More ports and expansion than Raspberry Pi or single-board solutions</li><li>Ideal for backups, Plex, file sharing, or surveillance recording</li></ul><h2>Minimum Hardware Requirements</h2><ul><li>1GHz x86 CPU (dual-core recommended)</li><li>2GB RAM minimum (4GB+ preferred)</li><li>1 or more SATA or USB drives</li><li>Ethernet port</li></ul><h2>Step 1: Prepare Installation Media</h2><ol><li>Go to <a href="https://www.openmediavault.org/" target="_blank">openmediavault.org</a> and download the latest ISO</li><li>Use <strong>Rufus</strong> (Windows) or <strong>Etcher</strong> (macOS/Linux) to flash the ISO to a USB drive</li></ol><h2>Step 2: Install OpenMediaVault</h2><ol><li>Boot your old PC from the USB drive (change boot order in BIOS if needed)</li><li>Select <strong>Install</strong> and follow the on-screen instructions</li><li>Choose the drive OMV should be installed on (typically an SSD or smaller HDD)</li><li>Set up a root password and allow installation to complete</li><li>After installation, remove the USB and reboot</li></ol><h2>Step 3: Initial Configuration</h2><ol><li>Connect the PC to your network via Ethernet</li><li>From another device, go to <code>http://&lt;OMV-IP&gt;/</code> in a browser</li><li>Login with <strong>admin</strong> / <strong>openmediavault</strong></li><li>Change the password and update the system</li></ol><h2>Step 4: Add Storage Drives</h2><ol><li>Go to <strong>Storage &gt; Disks</strong> and confirm your attached drives</li><li>Wipe the drives and create new filesystems (ext4 recommended)</li><li>Mount the filesystems and create shared folders</li><li>Enable SMB/CIFS (or NFS, FTP) under <strong>Services</strong> and assign shares</li></ol><h2>Tips & Tricks</h2><ul><li>Disable unneeded services to improve performance</li><li>Enable S.M.A.R.T. monitoring for disk health checks</li><li>Add Docker and Portainer for more advanced use</li></ul><h2>Troubleshooting</h2><ul><li><strong>No boot:</strong> Check USB boot priority or try using Legacy mode</li><li><strong>Drive not recognized:</strong> Use another SATA port or try wiping in BIOS</li><li><strong>Slow performance:</strong> Consider using an SSD for OMV system disk and upgrade RAM if possible</li></ul><h2>Conclusion</h2><p>OpenMediaVault is an excellent way to breathe new life into outdated hardware. With just a few steps, you can create a powerful and customizable NAS using a computer you already own. It’s perfect for home labs, media libraries, and backups — and it costs nothing but your time.</p>
