<h2>Introduction</h2><p>Syncthing is an open-source file synchronization tool that works similarly to Dropbox, but without the cloud dependency. It syncs files directly between your devices using encrypted peer-to-peer communication. In this guide, we’ll walk through how to install and configure Syncthing on your OpenMediaVault system.</p><h2>Step 1: Install OMV-Extras</h2><ol><li>Log into the OMV web interface.</li><li>Go to <strong>System &gt; Plugins</strong> and ensure <code>omv-extras.org</code> is installed. If not, download it from <a href="https://omv-extras.org" target="_blank">omv-extras.org</a> and upload it via the Plugins tab.</li></ol><h2>Step 2: Enable the Syncthing Plugin</h2><ol><li>Go to <strong>OMV-Extras</strong> in the left-hand menu.</li><li>Enable the <strong>Syncthing repo</strong> under the Extras section.</li><li>Navigate to <strong>Plugins</strong> again, search for "Syncthing," and click <strong>Install</strong>.</li></ol><h2>Step 3: Configure Syncthing</h2><ol><li>After installation, go to <strong>Services &gt; Syncthing</strong>.</li><li>Click <strong>Add</strong> to set up an instance.</li><li>Select the user account (e.g., <code>admin</code> or <code>pi</code>) to associate with Syncthing. This user will have access to the Syncthing web UI and control which folders are synced.</li><li>Choose a port (default: 8384) and data directory for synced files.</li><li>Click <strong>Save</strong> and then <strong>Enable</strong> and <strong>Start</strong> the service.</li></ol><h2>Step 4: Access Syncthing Web Interface</h2><p>Visit <code>http://your-server-ip:8384</code> in your browser. You’ll be taken to the Syncthing dashboard where you can begin configuring synced folders and devices.</p><h2>Step 5: Add Devices and Folders</h2><ul><li>Install Syncthing on another device (Windows, Mac, Linux, Android).</li><li>In the Syncthing web interface, click <strong>Add Remote Device</strong> and enter the Device ID of the other system.</li><li>On the second device, approve the pairing request.</li><li>Add a new folder on either device, then share it with the newly added device.</li><li>Both systems will sync the selected folder automatically going forward.</li></ul><h2>Tips & Tricks</h2><ul><li>Use versioning to retain copies of modified or deleted files under <strong>Folder Settings &gt; File Versioning</strong>.</li><li>Under <strong>Settings &gt; GUI</strong>, change the password to secure the web UI.</li><li>Use Syncthing’s ignore patterns to exclude specific files or folders from sync.</li><li>For advanced setups, use dynamic DNS and port forwarding for external access.</li></ul><h2>Troubleshooting</h2><ul><li><strong>Web UI won’t load?</strong> Ensure port 8384 is not blocked by a firewall.</li><li><strong>Files not syncing?</strong> Make sure both devices are online and folders are shared and accepted on both ends.</li><li><strong>Permissions errors?</strong> Ensure the Syncthing user has proper read/write access to the synced directories.</li></ul><h2>Conclusion</h2><p>Syncthing is a powerful, lightweight, and private alternative to cloud file sync services. Running it on OpenMediaVault ensures your files stay safe, up-to-date, and under your control across all your devices. Whether you're syncing between your NAS, desktop, or mobile device, Syncthing gets the job done securely and efficiently.</p>
