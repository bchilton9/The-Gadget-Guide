<h2>Introduction</h2><p>Syncthing is an open-source, decentralized file synchronization tool that works across all major platforms. This guide shows how to install and configure it on Windows, making it easy to sync files between your PC, mobile devices, and even Steam Deck or Raspberry Pi.</p><h2>Step 1: Download Syncthing for Windows</h2><ol><li>Visit the official Syncthing website: <a href="https://syncthing.net/downloads/" target="_blank">https://syncthing.net/downloads/</a></li><li>Download the latest 64-bit Windows build (ZIP file).</li><li>Extract the contents to a directory like <code>C:\Program Files\Syncthing</code>.</li></ol><h2>Step 2: Run Syncthing</h2><ol><li>Double-click <code>syncthing.exe</code>.</li><li>Allow through the Windows firewall if prompted.</li><li>Your default browser should open the Web UI at <code>http://localhost:8384</code>.</li></ol><h2>Step 3: Set Syncthing to Auto-Start</h2><ol><li>To make Syncthing run on login, create a shortcut of <code>syncthing.exe</code> in your Startup folder.</li><li>Press <kbd>Win + R</kbd>, type <code>shell:startup</code>, and press Enter.</li><li>Paste a shortcut to Syncthing there.</li></ol><p>Alternatively, install <a href="https://github.com/Martchus/syncthingtray" target="_blank">Syncthing Tray</a> for GUI control and system tray functionality.</p><h2>Step 4: Add Remote Devices</h2><ol><li>Open Syncthing's Web UI on both devices (e.g., Windows and Steam Deck).</li><li>Copy the Device ID from the <strong>Actions → Show ID</strong> window.</li><li>Paste it into the <strong>Remote Devices → Add Device</strong> section on the other device.</li><li>Give it a recognizable name and hit Save.</li></ol><h2>Step 5: Share Folders</h2><ol><li>Create or select a folder to share.</li><li>In the <strong>Shared With</strong> section, check the box for the remote device you just added.</li><li>Repeat the steps on the second device to accept and configure the shared folder.</li></ol><h2>Tips & Tricks</h2><ul><li>Use versioning (Settings → Versioning) to retain file history and prevent accidental loss.</li><li>Rename folders in Syncthing without changing the path on disk by editing the Folder Label field.</li><li>Use Sync Trayzor for easier management if you want a preconfigured Windows package.</li></ul><h2>Troubleshooting</h2><ul><li><strong>Can’t connect devices?</strong> Make sure they're both online and accessible, and the Device ID is correct.</li><li><strong>Firewall issues?</strong> Ensure Syncthing is allowed through the firewall or use relay servers under <strong>Settings → Connections</strong>.</li><li><strong>Web UI won’t open?</strong> Confirm Syncthing is running and visit <code>http://127.0.0.1:8384</code> manually.</li></ul><h2>Conclusion</h2><p>Syncthing is a robust, privacy-respecting sync solution for Windows users. With just a few setup steps, you can ensure your important folders are always backed up and synchronized across devices — without any cloud dependency.</p>
