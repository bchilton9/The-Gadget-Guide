<h2>Introduction</h2><p>The Steam Deck is more than a gaming device — it can be a powerful portable file sync station. This guide will show you how to install Syncthing and configure it to run in the background for hands-free synchronization on the Steam Deck.</p><h2>Step 1: Enter Desktop Mode</h2><p>Press the <strong>Steam</strong> button → <strong>Power</strong> → <strong>Switch to Desktop</strong>.</p><h2>Step 2: Install Syncthing</h2><ol><li>Open Konsole (from Application Launcher → System → Konsole).</li><li>Run the following commands:<pre>sudo pacman -Syu<br>sudo pacman -S syncthing</pre></li></ol><h2>Step 3: Create a Systemd User Service</h2><p>This ensures Syncthing runs automatically in the background:</p><pre>systemctl --user enable syncthing.service<br>systemctl --user start syncthing.service</pre><p>If you want Syncthing to start even when not logged into Desktop Mode, enable lingering:</p><pre>loginctl enable-linger $(whoami)</pre><h2>Step 4: Access Syncthing</h2><p>Open a browser and go to:</p><pre>http://localhost:8384</pre><p>This brings up the Syncthing Web UI where you can configure devices and folders.</p><h2>Step 5: Sync Your Files</h2><ol><li>Add your other devices (PC, phone, NAS) by exchanging Device IDs.</li><li>Create a folder to share and select which remote device(s) it should sync with.</li><li>Repeat on the other device to accept and configure the shared folder.</li></ol><h2>Tips & Tricks</h2><ul><li>Enable versioning on important folders to prevent accidental overwrites.</li><li>You can customize sync speed limits in the settings to preserve battery.</li><li>Use the <strong>Filesystem Watcher</strong> for real-time syncing (may impact battery life).</li><li>Install Syncthing Tray (AUR) for system tray access if using the Steam Deck frequently in Desktop Mode.</li></ul><h2>Troubleshooting</h2><ul><li><strong>Syncthing doesn’t start?</strong> Use <code>systemctl --user status syncthing</code> to debug logs.</li><li><strong>Syncing is slow?</strong> Check if the deck is in power-saving mode or if you're throttled by Wi-Fi.</li><li><strong>Folder won’t sync?</strong> Make sure folder IDs match and permissions are correct.</li></ul><h2>Conclusion</h2><p>With Syncthing running in the background, your Steam Deck becomes a capable sync client, allowing you to keep your ROMs, saves, and even documents synchronized across all your devices. The setup takes just a few minutes and runs quietly in the background without interrupting your gaming experience.</p>
