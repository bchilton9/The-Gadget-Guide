<h2>Introduction</h2><p>OpenMediaVault (OMV) offers a powerful interface to manage home backups. One effective strategy is to automate backups to an external USB drive â€” perfect for offline protection, archiving, or rotating disks for offsite storage.</p><h2>What You'll Need</h2><ul><li>OMV server or Raspberry Pi with USB ports</li><li>External USB hard drive or SSD</li><li>Mounted internal storage or NAS volume with your main data</li></ul><h2>Step 1: Mount the External USB Drive</h2><ol><li>Go to <strong>Storage &gt; Disks</strong> to confirm your USB drive is detected</li><li>Navigate to <strong>Storage &gt; File Systems</strong> and format or mount the drive</li><li>Once mounted, note the file system label for reference</li></ol><h2>Step 2: Create a Shared Folder on the USB Drive</h2><ul><li>Go to <strong>Access Rights Management &gt; Shared Folders</strong></li><li>Create a new folder such as <code>usb_backup</code> on the external drive</li><li>Set permissions appropriately (read/write for the system user that will run the backup)</li></ul><h2>Step 3: Set Up the Rsync Job</h2><ol><li>Navigate to <strong>Services &gt; Rsync &gt; Jobs</strong></li><li>Click <strong>Add</strong> to create a new rsync job</li><li>Choose the source folder (the main data share) and the destination (<code>usb_backup</code>)</li><li>Enable <strong>recursive</strong>, <strong>preserve permissions</strong>, and optionally <strong>delete</strong> (to sync deletions)</li></ol><h2>Step 4: Schedule the Job</h2><ul><li>Choose the frequency (daily, weekly, etc.)</li><li>Set the time when the USB drive is usually connected</li><li>Apply and enable the job</li></ul><h2>Step 5: Test and Monitor</h2><p>Manually run the job once to confirm functionality. You can check the status in the rsync log or through <strong>Diagnostics &gt; System Logs</strong>.</p><h2>Tips</h2><ul><li>Label USB drives with stickers to track rotation</li><li>Unmount drives when not in use to prevent wear</li><li>Store one USB drive offsite for disaster recovery</li></ul><h2>Conclusion</h2><p>Using external USB drives with rsync gives you an easy, flexible backup option for OpenMediaVault that doesn't rely on cloud or online sync tools.</p>
