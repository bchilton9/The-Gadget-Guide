<h2>Turn Your Raspberry Pi into a Retro Game Console</h2>

  <p>With a bit of software and a controller, your Raspberry Pi can play thousands of classic games from systems like NES, SNES, Genesis, PlayStation, and more. This guide shows how to install EmulationStation with RetroArch using a prebuilt image like RetroPie or Batocera.</p>

  <h3>What You Need</h3>
  <ul>
    <li>Raspberry Pi 3, 4, or 5</li>
    <li>16GB+ microSD card</li>
    <li>Game controller (USB or Bluetooth)</li>
    <li>HDMI display, keyboard for initial setup</li>
    <li>ROMs (legally backed up from your own games)</li>
  </ul>

  <h3>Option 1: Install RetroPie</h3>
  <ol>
    <li>Download the latest RetroPie image from <a href="https://retropie.org.uk/download/" target="_blank">retropie.org.uk</a></li>
    <li>Use Raspberry Pi Imager or Balena Etcher to flash the image to a microSD card</li>
    <li>Insert card into Pi and boot it up</li>
    <li>Configure your controller and Wi-Fi on first boot</li>
  </ol>

  <h3>Option 2: Try Batocera</h3>
  <p>Batocera is a user-friendly alternative with built-in game scraping and beautiful themes.</p>
  <ol>
    <li>Download Batocera from <a href="https://batocera.org/" target="_blank">batocera.org</a></li>
    <li>Flash it using Balena Etcher</li>
    <li>Boot your Pi with Batocera and configure input devices</li>
  </ol>

  <h3>Transferring ROMs</h3>
  <ul>
    <li>RetroPie: Copy via USB (plug in a blank USB drive to create folder structure)</li>
    <li>Batocera: Transfer over network using SMB shares</li>
  </ul>

  <h3>Systems Supported</h3>
  <p>RetroPie and Batocera support hundreds of consoles and handhelds, including:</p>
  <ul>
    <li>NES, SNES, Sega Genesis, Master System</li>
    <li>PlayStation 1, PSP, Dreamcast</li>
    <li>Game Boy, GBA, N64 (varies by model)</li>
  </ul>

  <h3>Controller Recommendations</h3>
  <ul>
    <li>8BitDo SN30 Pro (Bluetooth or USB)</li>
    <li>Xbox controllers (wired or wireless with dongle)</li>
    <li>PS4/PS5 via Bluetooth</li>
  </ul>
